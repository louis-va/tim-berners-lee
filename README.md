# Getting Started

## Prerequisites

You need to have node and npm installed on your machine to run this project locally.  

### npm
  ```sh
  npm install npm@latest -g
  ```

## Installation

### 1. Clone the repo
```sh
git clone https://github.com/your_username_/Project-Name.git
```
### 2. Install NPM packages
```sh
npm install
```

## Execution 
```sh
npm run dev
```
